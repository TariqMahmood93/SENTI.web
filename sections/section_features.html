Features

Key Capabilities

SENTI combines state-of-the-art NLP with practical data engineering to deliver a complete imputation workflow.

🎯
Zero-Shot Operation
No fine-tuning or labeled imputation examples required. Works out-of-the-box on any relational dataset with pre-trained transformer models.
⚡
Incremental Mode
Efficient online imputation for streaming data. The FAISS index is updated incrementally, enabling constant-time imputation for new tuples.
🔬
Multiple Null Patterns
Supports MCAR, MAR, and MNAR missing data generation with configurable rates, target columns, and conditions for realistic evaluation.
📊
Built-in Evaluation
Comprehensive metrics including accuracy, F1-score, precision, recall, and RMSE for categorical and numerical columns respectively.
🌐
Multilingual Support
LaBSE sentence transformer enables imputation across multilingual datasets, handling heterogeneous text in multiple languages seamlessly.
🖥️
Interactive UI
Full Streamlit web application with visual diff highlighting of imputed cells, dataset statistics, and downloadable results in multiple formats.
