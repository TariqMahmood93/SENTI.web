What is SENTI?

SENTI (SENTI-based Data Imputation) is a data imputation system that uses sentence transformer embeddings to semantically recover missing values in incomplete relational tables — without any model training or labeled data.

Imputation Pipeline
1
Incomplete Dataset R₀
Input with missing values (⊥)
2
Sentence Embedding
Encode tuples via SENTI / LaBSE
3
Semantic Similarity Search
FAISS k-NN over embedded space
4
Value Propagation
Majority vote / weighted imputation
5
Complete Dataset R₀*
Imputed output ready for downstream use
🎯
Zero-Shot Imputation
No pre-training or labeled missing data required. SENTI leverages pre-trained sentence transformers to impute values purely from semantic context.
📐
Relational Semantics
Treats tuples as natural language sentences, capturing inter-attribute relationships that tabular models miss.
⚡
Incremental Support
Handles streaming data with online index updates — new tuples are imputed without reprocessing the entire dataset.
🔬
Evaluation Suite
Built-in metrics (accuracy, F1, RMSE) compare imputed vs. ground truth, with support for multiple null injection patterns (MCAR, MAR, MNAR).
