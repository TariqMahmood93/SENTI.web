How to Use SENTI

Run SENTI locally in three steps. Requires Python 3.9+ and pip.

01 Clone & Install 
# Clone the repository
git clone https://github.com/TariqMahmood93/SENTI
cd senti

# Install dependencies
pip install -r requirements.txt
02 Launch the App 
# Start the Streamlit interface
streamlit run app.py

# App opens at http://localhost:8501
03 Run via Python API 
# Static imputation
from senti import SENTIImputer

imputer = SENTIImputer(model="all-MiniLM-L6-v2", k=5)
imputed_df = imputer.fit_transform(df_with_nulls)

# Incremental imputation
imputer.update(new_batch_df)
imputed_new = imputer.transform(new_batch_df)
